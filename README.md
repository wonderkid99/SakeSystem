# S.A.K.E (Sistem Audit dan Kesejahteraan Karyawan)

![Language](https://img.shields.io/badge/Language-C%23-blue)
![Platform](https://img.shields.io/badge/Platform-.NET%208.0-purple)

Sebuah aplikasi console sederhana berbasis C# untuk mengaudit kinerja lembur bulanan dan menghitung tunjangan bonus kesejahteraan karyawan di PT. Cepat Sampai.

> **Catatan:** > Proyek ini dibuat untuk memenuhi tugas mata kuliah **Pemrograman Lanjut**.

Sistem ini dirancang dengan logika pemrograman dasar yang mencakup:
1.  **Looping (Perulangan):** Input data karyawan secara berulang hingga user mengetik perintah 'EXIT'.
2.  **Switch-Case:** Penentuan tunjangan dasar berdasarkan posisi jabatan (Manajer, Staf Senior, Staf Junior, dll).
3.  **Complex If-Else:** Perhitungan bonus lembur bertingkat (Sangat Loyal, Cukup Loyal, Normal, Nihil).
4.  **Simple If:** Indikator kinerja prima jika total pendapatan melebihi ambang batas tertentu.
